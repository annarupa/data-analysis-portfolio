# Data Analysis Portfolio – Anna Rupa Anthony

Production Support | Data Quality | SQL | Python | Tableau

This repository contains sample scripts and queries demonstrating skills in:
- SQL-based data investigation and root-cause analysis
- Data quality validation and reconciliation
- Python data analysis and reporting automation
- Tableau dashboard data preparation

These samples reflect real-world patterns from enterprise production support environments across **financial services** (incentive & partner programs) and **healthcare** (customer service operations & SLA monitoring) domains.

---

## Files

| File | Description |
|------|-------------|
| `01_root_cause_investigation.sql` | Financial: Investigating timing discrepancies between transaction and settlement dates |
| `02_data_quality_checks.sql` | Reusable data quality rules: nulls, duplicates, referential integrity, outliers, volume trends |
| `03_reconciliation_report.sql` | End-to-end reconciliation between source (EDL) and target reporting system |
| `04_data_analysis_python.py` | Python script for automated data profiling and anomaly detection across both domains |
| `05_tableau_prep_query.sql` | Optimized query for Tableau dashboard data prep — financial incentive performance |
| `06_healthcare_sla_investigation.sql` | Healthcare: CRM-to-HPOM data flow validation and customer case SLA breach analysis |
